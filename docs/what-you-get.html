Knowledge Graph

A local SQLite graph that grows automatically from your work. No external infrastructure — just Python + SQLite, works offline.

ComponentWhat It DoesHow It Helps
graph-auto-index.py hookEvery Write/Edit indexes entities, computes relationship edgesGraph builds itself — zero maintenance
/graph-query skillQuery relationships, find connections, discover related filesSurface context you didn't know existed
wiki/entities/Company, product, concept pages indexed by the graphStructured knowledge for traversal
wiki/people/Person pages with org trees and interaction timelinesRelationship intelligence that compounds

How it compounds: Write about a deal. The hook indexes the account, people, and products mentioned. Next time you prep for that account, the graph surfaces related wiki pages, other deals with the same people, and memory files you'd forgotten. The more you write, the smarter retrieval gets.

The Compound Effect

No single component is transformative alone. The value is in how they interact:

Day 1:  You correct Claude → Memory file saved
Day 3:  Same mistake class → Rule prevents it automatically
Day 7:  Rule forgotten in long session → Hook enforces mechanically
Day 14: Hook catches a pattern → /skillify extracts it into a new command
Day 30: Overnight crons feed intel → Morning brief surfaces it
Day 60: Graph connects entities → Related context surfaces automatically

Starting matters more than perfecting. A mediocre setup that runs for 60 days beats a perfect setup that runs for 1 day. Compound growth is the product. The kit gives you Day 5 — compound loops take over from there.
